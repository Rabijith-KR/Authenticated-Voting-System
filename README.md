# Authenticated Voting System Using RFID and Fingerprint Authentication

## Overview
This project implements a secure electronic voting system that combines RFID technology and fingerprint authentication to ensure voter identity verification and prevent duplicate voting.

## Features
- Voter Registration
- RFID Card Authentication
- Fingerprint Enrollment
- Fingerprint Verification
- Secure Vote Casting
- Vote Storage Using EEPROM
- Election Result Display

## Hardware Components
- Arduino Mega 2560
- MFRC522 RFID Reader
- Fingerprint Sensor
- LCD I2C Display
- EEPROM
- Push Buttons
- Buzzer

## Programming Language
- C++ (Arduino Framework)

## Team Project
Developed as a group project for implementing a secure and user-friendly electronic voting system.
